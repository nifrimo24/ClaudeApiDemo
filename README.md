# ClaudeApiDemo

## Setup

1. Clona el repo.
2. Copia el archivo de ejemplo de `launchSettings.json`:

   ```powershell
   copy ClaudeApiDemo\Properties\launchSettings.json.example ClaudeApiDemo\Properties\launchSettings.json
   ```

3. Abre `ClaudeApiDemo\Properties\launchSettings.json` y sustituye `TU_API_KEY_AQUI` por tu API key de Anthropic.
4. Ejecuta el proyecto normalmente (F5 / `dotnet run`).
